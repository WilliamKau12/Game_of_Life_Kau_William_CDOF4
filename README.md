# Game_of_Life_Kau_William_CDOF4
The game of life using spaces for dead cells and # for live cells
Some documentation about the [rules](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life).


To run [Game of Life](http://www.conwaylife.com/wiki/Conway%27s_Game_of_Life):
```bash
$ python conway.py
```
## Goals 
The goal of this project is to code the game of life and study its patterns.
